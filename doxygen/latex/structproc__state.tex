\doxysubsection{Detailed Description}
Represents a complete snapshot of the processor\textquotesingle{}s general-\/purpose register state when in x86-\/64 mode. This structure is used by functions such as \mbox{\hyperlink{cpuid_8c_ad9b15889d2f56638808ea71ce96ecfc7}{\+\_\+cpuid}} to avoid having to convolve registers in other ways. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{cpuid_8c_ad9b15889d2f56638808ea71ce96ecfc7}{\+\_\+cpuid}} 
\end{DoxySeeAlso}
